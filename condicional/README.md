```
- if-else.c
```

If and Else em C

```c
if(boolean){}
else {}
```

```
- ternary.c
```

Operador ternário em C

```c
(boolean)?command_1:command_2;
```

```
- switch.c
```

Switch-case em C

```c
switch(var):
    case 1:
        break;
    default:
        break;
```

* Para manipular bem essas funções é bom ter o conhecimento de operadores lógicos!

- && -> and
- || -> or
- ! -> not

* Exemplos:

```c
if(a && b)
else if(!a)
else if(a || b)
```
Sendo a e b valores booleanos(zero ou diferente de zero)
